# Il Segreto del Reattore Perduto
 
Questo progetto è un mini gioco RPG testuale ispirato al mondo di Iron Man and His Awesome Friends.  
Il giocatore prende decisioni durante la storia che influenzano lo svolgimento degli eventi e il finale.  
Serve per simulare un sistema di gioco a scelte con combattimenti, personaggi e finali multipli.
Ho utilizzato CHATGPT per chiarire errori di compilazione.

Il gioco è basato su:
- Scelte multiple
- Eventi narrativi
- Combattimenti (se presenti).

Tecnologie utilizzate :
- Java
- Gradle
- Git / GitHub
Come eseguire il progetto:

Da terminale:
```bash
./gradlew run
