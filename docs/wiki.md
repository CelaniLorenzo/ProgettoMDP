# Il Segreto del Reattore Perduto

## Descrizione del progetto
Questo progetto è un mini gioco RPG testuale ispirato al mondo di Iron Man.
Il giocatore prende decisioni che influenzano la storia e portano a finali diversi.

## Gameplay
Il gioco è basato su:
- Scelte multiple
- Eventi narrativi
- Combattimenti (se presenti)

Ogni decisione cambia il corso della storia.

## Struttura del progetto
Il progetto è organizzato in:

- `app/src/main/java` → codice principale
- `app/src/test/java` → test
- `docs/` → documentazione (questa wiki)

## Tecnologie utilizzate
- Java
- Gradle
- Git / GitHub
